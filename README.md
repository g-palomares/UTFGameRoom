# UTFGameRoom

COMO RODAR O PROJETO?

Além dos requisitos mínimos básicos como um ambiente PHP com XAMPP, será necessário clonar o repositório (git clone https://github.com/g-palomares/UTFGameRoom.git) efetuar a 
instalação do Composer, digitando composer install na linha de comando do VSCode enquanto estiver no local do projeto onde se encontra o arquivo composer.json.
